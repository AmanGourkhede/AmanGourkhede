## 🚀 Projects

### ⭐ Multi-Agent Deep Research Engine
> 7-phase AI research pipeline built entirely from scratch — no LangChain, no paid APIs. Every claim is verified before it reaches the writer.

[![multi-agent-deep-research](https://github-readme-stats.vercel.app/api/pin/?username=AmanGourkhede&repo=multi-agent-deep-research&theme=tokyonight&hide_border=true&bg_color=0D1117&title_color=6E57F7&icon_color=6E57F7)](https://github.com/AmanGourkhede/multi-agent-deep-research)

| Phase | What Happens |
|-------|-------------|
| 🌐 Scrape | 4 search queries × 5 pages via BeautifulSoup |
| 🔍 Extract | Individual factual claims pulled from each page |
| ✅ Verify | LLM scores each claim against evidence — 65% confidence threshold |
| 🎯 Judge | Relevance scored against topic — 60% threshold, sorted best-first |
| 📊 Diagram | AI finds numerical data → Plotly renders PNG charts |
| ✍️ Write | 7-section 2000+ word article from verified facts only |
| 🧑‍⚖️ Edit | Editor agent fixes weak sections, improves conclusion |
| 📄 Export | Self-contained HTML with embedded charts + PDF |

`Python` `Ollama` `ChromaDB` `Redis` `Pydantic v2` `Plotly` `BeautifulSoup` `Streamlit` `Docker`

### 📚 RAG From Scratch
> Complete Retrieval-Augmented Generation pipeline with zero framework dependencies — every component hand-built and explained.

[![rag-from-scratch](https://github-readme-stats.vercel.app/api/pin/?username=AmanGourkhede&repo=rag-from-scratch-&theme=tokyonight&hide_border=true&bg_color=0D1117&title_color=6E57F7&icon_color=6E57F7)](https://github.com/AmanGourkhede/rag-from-scratch-)

| Component | Role |
|-----------|------|
| Loader | PDF + TXT ingestion via PyMuPDF |
| Chunker | Overlapping text segmentation |
| Embedder | `all-MiniLM-L6-v2` via HuggingFace |
| VectorStore | ChromaDB indexing and persistence |
| Retriever | Semantic nearest-neighbour search |
| Generator | Ollama LLM with retrieved context |
| Evaluator | RAGAS — faithfulness, relevancy, precision |

`Python` `Ollama` `ChromaDB` `sentence-transformers` `HuggingFace` `RAGAS` `PyMuPDF`
